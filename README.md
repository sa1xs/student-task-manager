# Student Task Manager

Una aplicación de gestión de tareas sencilla y moderna construida con **HTML**, **CSS** y **JavaScript**, ideal para que los estudiantes organicen su día a día de manera rápida.

## Características
- **Interfaz Limpia y Moderna:** Diseño con alto contraste que facilita la concentración.
- **Tipografía Legible:** Integración de la fuente *Inter* para una experiencia de lectura muy agradable.
- **Gestión Rápida:** Escribe tu pendiente en el input y presiona el botón para mantener tu lista actualizada al instante.

## Tecnologías Utilizadas
- **HTML5:** Estructura de documentos.
- **CSS3 Vanilla:** Estilos construidos desde cero, aprovechando *flexbox*, transiciones y un diseño personalizado.
- **Vanilla JavaScript:** Manipulación directa del DOM para agregar elementos rápidamente sin necesidad de recargar la página.

## Cómo usar

1. Clonar el repositorio o descargar los archivos
2. Abrir `index.html` en cualquier navegador
3. No requiere instalación ni dependencias
